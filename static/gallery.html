Additional Message

GroupNotes is inspired by PLATO Notes, the communication system originally created by David Woolley in 1973, which was the precursor to today's web-based message boards.

While I've never used the original PLATO Notes, I remember using VAX/VMS Notes back in the early 90's. It was amazing. You can post messages, often a question about something, and other people on the Notes system can reply to it in something resembling a message thread. This was pre-Web and Internet.

Today (I'm writing this in 2020), we have many forms of disussion groups and messageboards on the web. They all work well and serve its purpose. But I wanted to create something that came closer to the spirit of the original PLATO Notes, while at the same time embracing modern web technologies and interfaces. GroupNotes came about from that.
